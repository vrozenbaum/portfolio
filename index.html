US Exonerations

Working with the National Registry of Exonerations Database, this project was heavily focused on exploratory data analaysis and inference. I was interested in looking at exoneration trends over the years as well as racial disparities in the data. I conducted Bayesian Inference to test whether post 1994 black and white murder exonerees were statistically different. The modeling portion consisted of building a panel regression which predicted exonerations per state for 2017.

Web Scraping Indeed.com and Salary Prediction

This was a multipart project that involved building a webscraper for Indeed.com that returned listings for Data Scientist jobs. After the listings were gathered, I built multiple models that were able to predict whether a position's salary would fall below or above the median based on the previously extracted features.

Global Terrorism Database: Bayesian Inference and Imputation

This project involved using the Global Terrorism Database to conduct Bayesian Inference on two different groups to test whether they were statistically different. Since the database is missing data from 1993, we were asked to impute the total number of bombings that occured that year.
